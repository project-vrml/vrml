# vrml-spi

Java SPI工具集。

## Usage

### Maven

```xml

<dependency>
    <groupId>group.rxcloud</groupId>
    <artifactId>vrml-spi</artifactId>
    <version>1.0.5</version>
</dependency>
```

### SPI

Java SPI加载器。

#### 使用

引入`vrml-spi`依赖包即可。

#### SPI加载

```java
SPI.loadXXX()
```
