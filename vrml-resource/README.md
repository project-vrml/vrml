# vrml-resource

Java File工具集。

## Usage

### Maven

```xml

<dependency>
    <groupId>group.rxcloud</groupId>
    <artifactId>vrml-resource</artifactId>
    <version>1.1.0</version>
</dependency>
```

### Resources

Java Resources文件加载器。带有线程安全的缓存效果。

#### 使用

引入`vrml-resource`依赖包即可。

#### File资源加载

```java
Resources.loadXXX()
```
